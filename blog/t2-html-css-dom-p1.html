What is boxifying?

Boxifying is reducing all webpage content to there own boxes. Basically everything you see on a webpage can be reduced to a box big or small.

What is the box model?

The box model is in reference to design and layout. It is essentially a box that wraps around all HTML elements. It consists of: margins, boarders, padding and all the actual content.
