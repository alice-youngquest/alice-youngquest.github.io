But this isnt really something iv tried doing everyday... so having done this for the last week has been different! First off i can say it was a struggle to actually make myself do it, i would keep putting it off cause i felt like i had no reason to do it. So this is where i needed to change my mindset i decided id do it at the sametime every day i choose 9pm cause im always definitely home then. I realised that having this 2minutes set aside everyday just to breathe and apreciate made me calmer as a person. Im laughing at myself right now cause i never thought id say this but it really made me appreciate everything that i had done or seen that day and i felt happier for it. I think this is definitely something i want to carry on with.
